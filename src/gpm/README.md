# gpm Package

Reusable Python utilities for the groundwater prospectivity POC.

Modules:

```text
paths.py         shared repository paths
gee_auth.py      Earth Engine service-account initialization
gee_features.py  leakage-safe GEE feature extraction
progress.py      compact terminal progress indicator
batch_merge.py   batch parquet merge and deduplication
modeling.py      XGBoost training, tuning, evaluation, artifact saving
```
